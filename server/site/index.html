How to verify

You can verify any contribution by downloading the required files and using a command line instruction to run the verification

    Prerequisites
  • Node.js, version 12 or higher
  • Install snarkjs: 
    npm install -g snarkjs

    Commands
  • Download the Powers-of-Tau file (see circuit page to find which is required)
  • Download the contribution zkey file you wish to verify

    • then

  • Download the circuit's .r1cs file
  • Run the verification command: 
    snarkjs zkv [.r1cs] [.ptau] [.zkey]

    • or

  • Download the initial zkey file (contribution 0)
  • Run the verification command: 
    snarkjs zkvi [cxx_0000.zkey] [.ptau] [.zkey]

A successful verification will end with: 

[INFO]  snarkJS: ZKey Ok!
